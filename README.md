### Задание 1

`Установите Docker Compose и опишите, для чего он нужен и как может улучшить вашу жизнь`

1. `Docker Compose упрощает задачи, связанные с Docker, за счет возможности определять и запускать многоконтейнерные приложения с помощью одного YAML-файла. Это позволяет легко управлять зависимостями между контейнерами, их сетевыми настройками и общими конфигурациями, что значительно упрощает процесс развертывания и масштабирования приложений. Используя команды docker-compose up и docker-compose down, можно быстро запускать и останавливать весь набор контейнеров, что делает управление инфраструктурой более эффективным и удобным.`
